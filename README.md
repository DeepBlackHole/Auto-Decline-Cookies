### What script do 
This script automates the decline of cookies.
### How it works
Finds elements like < button >, < a >, < input > with text:

* "Reject all"
 
* "Decline"

* "Only essential"

* "Continue without accepting"

* "Do not accept"

* "Refuse"

Click on them if found

Repeat and check every 15 seconds
